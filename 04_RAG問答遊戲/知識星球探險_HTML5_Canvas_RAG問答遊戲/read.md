# 知識星球探險 HTML5 Canvas RAG 問答遊戲

## 分類

RAG 問答遊戲

## 執行方式

直接用瀏覽器開啟本資料夾內的 `index.html`。

## 操作方式

- `Enter`：開始遊戲
- `WASD` 或方向鍵：操作太空船
- 靠近星球後按 `E`：進入知識站
- 知識站中按 `Enter`：查詢問題
- 知識站中按 `Esc`：離開知識站

## 遊戲說明

玩家駕駛太空船前往不同知識星球。每個星球代表一個主題，例如 AI、環保、校園安全、程式設計。玩家輸入問題後，系統會從 JavaScript 內建知識庫進行關鍵字比對與相關度計分，顯示來源、相關度與整理後回答。

## 原始提示詞

請製作一個可直接執行的 HTML5 Canvas 知識問答遊戲，遊戲名稱為「知識星球探險」，模擬簡易 RAG 知識庫檢索流程。玩家駕駛太空船前往 AI、環保、校園安全、程式設計等知識星球，進入星球後可輸入問題。使用 JavaScript 陣列建立內建知識庫，每筆資料包含主題、標題、關鍵字與內容，根據玩家問題進行關鍵字比對與相關度計分，顯示來源名稱、檢索結果與整理後回答。完成四個星球指定問題後顯示「知識探險家」。所有程式碼放在單一 `index.html`。
