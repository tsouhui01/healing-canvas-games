# 發布到 GitHub Pages 操作紀錄

## 目前狀態

本資料夾已整理成 GitHub Pages 可直接發布的結構：

- 根目錄 `index.html`：療育遊戲區入口頁
- `01_射擊遊戲`～`07_節奏反應遊戲`：七個遊戲分類資料夾
- 每個遊戲資料夾內都有 `index.html` 與 `read.md`
- 本資料夾已完成 Git 初始化與第一次 commit

## 建議 Repository 名稱

`healing-canvas-games`

## 建立 GitHub Repository

1. 開啟 `https://github.com/new`
2. Repository name 輸入 `healing-canvas-games`
3. 建議選擇 `Public`
4. 不要勾選初始化 README、`.gitignore` 或 License
5. 按 `Create repository`

## 建立後

請把 repository 網址提供給 Codex，例如：

`https://github.com/你的帳號/healing-canvas-games`

Codex 會接著協助加入遠端、推送 `main` 分支，並指導開啟 GitHub Pages。
