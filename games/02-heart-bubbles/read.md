# 心光泡泡 HTML5 Canvas 情緒療癒遊戲

## 分類

情緒療癒遊戲

## 執行方式

直接用瀏覽器開啟本資料夾內的 `index.html`。

## 操作方式

- `Enter`：開始或重新開始遊戲
- `WASD` 或方向鍵：移動愛心
- `Space`：釋放溫柔脈衝，推開附近哭臉泡泡

## 遊戲說明

玩家控制愛心，在彩虹背景中收集笑臉泡泡並避開哭臉泡泡。遊戲包含柔和動畫、漂浮泡泡、生命 3 條與分數系統。

## 原始提示詞

請製作一個 HTML5 Canvas 情緒療癒遊戲。玩家控制愛心，收集笑臉泡泡，避開哭臉泡泡，背景彩虹，泡泡漂浮，柔和動畫，生命 3 條，分數系統，Enter 開始。使用純 HTML、CSS、JavaScript，所有程式碼放在單一 `index.html`。
