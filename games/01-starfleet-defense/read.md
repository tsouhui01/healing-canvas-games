# 星際防線 HTML5 Canvas 射擊遊戲

## 分類

射擊遊戲

## 執行方式

直接用瀏覽器開啟本資料夾內的 `index.html`。

## 操作方式

- `Enter`：開始或重新開始遊戲
- `WASD` 或方向鍵：移動太空船
- `Space`：發射子彈

## 遊戲說明

玩家控制太空船，在深色太空背景中射擊敵人。敵人會持續從上方出現，擊中敵人可得分，碰到敵人會扣生命。玩家共有 3 條生命，分數越高敵人速度越快。

## 原始提示詞

請製作一個可直接執行的 HTML5 Canvas 射擊遊戲。玩家控制太空船或戰鬥機，使用鍵盤方向鍵或 WASD 移動，空白鍵發射子彈，Enter 開始或重新開始。敵人會不斷出現，擊中敵人可得分，碰到玩家扣生命，生命歸零顯示 Game Over。畫面需有深色太空背景、星星移動效果、分數、生命值與科技感 UI。使用純 HTML、CSS、JavaScript，不使用外部套件與圖片素材，所有程式碼放在單一 `index.html`。
