# 星光音樂盒｜音符創作遊戲

## 分類

音樂創作遊戲

## 執行方式

直接用瀏覽器開啟本資料夾內的 `index.html`。

## 操作方式

- `Enter`：開始或重新開始
- 滑鼠點擊音符星球：播放並記錄音符
- 數字鍵 `1`～`8`：播放並記錄對應音符
- `Space`：播放已記錄的旋律
- `Backspace`：刪除最後一個音符
- `C`：清除旋律

## 遊戲說明

畫面中有 8 個不同顏色的音符星球。玩家可以點擊星球或按數字鍵創作旋律，系統會使用 Web Audio API 即時產生聲音，不載入任何外部音樂檔案。最多可記錄 32 個音符，完成 4 個以上音符即可播放完整旋律。

## 原始提示詞

請製作一個可直接執行的 HTML5 Canvas 音樂創作遊戲，遊戲名稱為「星光音樂盒」。畫面中排列八個不同顏色的音符星球，點擊不同星球會播放不同音高，鍵盤數字鍵 `1`～`8` 對應八個音符。按空白鍵播放已記錄的旋律，按 `Backspace` 刪除最後一個音符，按 `C` 清除旋律，按 `Enter` 開始或重新開始。使用 Web Audio API 即時產生聲音，不使用外部套件、音樂檔案或圖片素材，所有程式碼放在單一 `index.html`。
