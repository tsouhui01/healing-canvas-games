# 彩虹節奏島｜節奏反應遊戲

## 分類

節奏反應遊戲

## 執行方式

直接用瀏覽器開啟本資料夾內的 `index.html`。

## 操作方式

- `Enter`：開始或重新開始遊戲
- `D`、`F`、`J`、`K`：控制四條音軌
- `Space`：暫停或繼續
- 畫面按鈕：切換簡單、普通、困難

## 遊戲說明

彩色音符會由畫面上方往下移動，玩家需在音符進入底部判定線時按下對應按鍵。Perfect 獲得 100 分，Good 獲得 50 分，Miss 會中斷連擊。遊戲使用 Web Audio API 產生簡單節拍音效，不使用外部音樂素材。

## 原始提示詞

請製作一個可直接執行的 HTML5 Canvas 節奏遊戲，遊戲名稱為「彩虹節奏島」。彩色音符由畫面上方往下移動，玩家需在音符進入判定區時按下對應按鍵。設有四條音軌，使用 `D`、`F`、`J`、`K` 控制，空白鍵暫停或繼續，Enter 開始或重新開始。Perfect 100 分，Good 50 分，Miss 中斷連擊，顯示分數、連擊數、最高連擊與評級。使用 Web Audio API 產生節拍音效，純 HTML、CSS、JavaScript，不使用外部套件、圖片或音樂素材，所有程式碼放在單一 `index.html`。
